========================================
TROUBLESHOOTING:
========================================

If you get "Can't reach database server":
- Check DATABASE_URL has port 6543 (not 5432)
- Check password is URL-encoded
- Check ?pgbouncer=true is at the end

If you get "Tenant or user not found":
- Your database password is incorrect
- Get the correct password from Supabase dashboard

If you get "NO_SECRET" error:
- Check NEXTAUTH_SECRET is set in Vercel

If Google OAuth doesn't work:
- Check GOOGLE_CLIENT_ID and GOOGLE_CLIENT_SECRET are set
- Check redirect URI is configured in Google Cloud Console
